# 🤖 AgenTank Raid Helper (坦克出击 Raid 挂机助手)

[![Version](https://img.shields.io/badge/version-2.2.12-blue.svg)](https://github.com/xiedaibin/agentank-raid-helper)
[![Platform](https://img.shields.io/badge/platform-Chrome%20Extension%20v3-green.svg)](https://chrome.google.com/)

**AgenTank Raid Helper** 是一款专门针对 `https://agentank.ai/raid` 出击模式设计的智能全自动挂机 Chrome 浏览器插件（基于 Chrome Extension Manifest V3 标准开发）。

本插件旨在通过高内聚的自适应状态机引擎与智能决策算法，实现全自动出击、智能坦克选择、强化技能精准抉择、战术撤离锁分、自动仓库售宝和星屑低风险兑换星星的闭环无限循环挂机。

> 🎯 **终极目标**：智能低风险锁分挂机上王者，胜率达到 55% 以上。

---

## ✨ 核心功能与特性

### 1. 📊 玻璃拟态侧边栏双 Tab 仪表盘
为了解决由于功能增多导致面板内容过长、挤压日志和统计视图的痛点，侧边栏采用了精美的分栏 Tab 交互设计：
*   **常驻头部卡片**：控制台**主开关**与**当前状态**（正在出击/未运行）常驻顶部，无论切换到哪个 Tab，核心开关与状态都触手可及。
*   **📊 运行监控 Tab**：
    *   **实时状态**：当前运行阶段、当前层数、星星与星屑余额。
    *   **强化效果**：实时展示当前这一轮（Run）中已获得的强化技能与对应等级。
    *   **战绩统计**：详细记录运行时长、累计点击数、出击次数、胜利层数以及撤离与失败次数。
    *   **操作日志**：高达 `150px` 的滚动日志框，实时输出带有彩色级别标识（信息/警告/错误/操作/状态）的运行日志。
*   **⚙️ 挂机设置 Tab**：
    *   **撤离设置**：支持自定义阶段一、阶段二和最大强制撤离层数。
    *   **技能配置**：支持配置“快速装填”最大获取次数。
    *   **二次升级优先级**：支持拖拽胶囊标签，实时重排辅助芯片的升级顺序。
*   **💾 配置持久化与 Tab 记忆**：所有修改过的参数以及用户最后停留的 Tab 状态（`sidebarActiveTab`）均会自动保存至 `chrome.storage.local`，刷新或重启浏览器后无缝恢复。

### 2. 🛡️ 智能战术撤离算法 (Evacuation Strategy)
根据当前局内胜场与所拥有的复活命数进行低风险收益最大化评估，支持精细化的三档层数自定义配置：
*   **层数限制判定**：
    *   **≤ 阶段一结算层**（默认 3 层）：无条件**继续**挑战，快速累积初始战斗力。
    *   **阶段一 ➔ 阶段二**（默认 3-5 层）：若此时拥有的 `备用核心 == 0`，则在挑战胜利时**撤离并保存**；若 `备用核心 >= 1`，则**继续**挑战。
    *   **阶段二 ➔ 最大限制层**（默认 5-7 层）：若拥有的 `备用核心 < 2`，则挑战胜利时**撤离并保存**；若 `备用核心 >= 2`，则**继续**挑战。
    *   **≥ 最大强制撤离层**（默认 7 层）：挑战完成后**无条件撤离并保存**（保守锁分，落袋为安）。
*   **避让撤离机制**：当结算关卡正好处于 3 层或 5 层，但奖励三选一中刷新了“备用核心”时，决策算法会自动判定潜在核心数，**优先选择备用核心并继续挑战**，避免因误撤离而浪费绝佳起胡局。

### 3. 🔮 强化技能智能选择策略 (Upgrade Priority)
当胜利并出现三选一强化奖励时，状态机将按照以下决策链进行优先级评估：
1.  **绝对第一优先级**：`备用核心`（只要出现且未满 Lv.2，必选它）。
2.  **绝对第二优先级**：`快速装填`（根据用户在设置中填写的最大次数限制 `rapidReloadMax` 决定，默认 2 次。在此限额内作为第二优先级必选；超出限额后，自动退回为常规辅助技能，避免挤占防御技能位置）。
3.  **首次拥有优先**：其他核心芯片未获得过（Lv.0）的，优先各获得一次（`自动护盾`、`宝物磁场`、`技能冷却`、`开局推进`）。
4.  **二次升级优先级**：在均已获得后，按照用户自定义的排序顺序依次进行升级。支持在侧边栏或 Popup 页面通过按住 `☰` **原生拖拽（Drag & Drop）** 随时重排（默认顺序为：`自动护盾` > `宝物磁场` > `技能冷却` > `开局推进`）。

### 4. 💰 仓库资产自动流转
*   **一键出售**：自动检测仓库页面，依次点击 “全部出售” 或 “出售” 按钮，秒清所有挂机产出的宝物，变现为星屑。
*   **星屑兑换星星**：当大厅内 `星星 == 0` 且 `星屑 > 0` 时，自动进入仓库在兑换框内输入 `"1"` 并点击 `"兑换星星"`。按需小额兑换，防止一次性将星屑全部兑换为星星，有效分散游戏内星屑的持有风险。

## 🚀 安装与使用说明

### 1. 本地加载插件
1. 下载或克隆本项目到本地。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/` 进入扩展程序管理页面。
3. 开启右上角的 **“开发者模式”** 开关。
4. 点击左上角的 **“加载已解压的扩展程序”** 按钮，选择本项目所在的根目录。
5. 看到具有赛博坦克图标的 **Agentank Raid Helper** 插件即表示加载成功。

### 2. 挂机运行
1. 打开挂机目标网页：[https://agentank.ai/raid](https://agentank.ai/raid)。
2. 页面右侧会自动展开玻璃拟态的控制面板侧边栏。
3. 确保侧边栏上的 **“主开关”** 处于开启状态，插件将自动检测当前页面状态并开始全自动挂机。
4. 如果需要暂停，随时在侧边栏或点击插件右上角图标关闭主开关即可。

### 3. 常规日志屏蔽与开启
*   如果您希望在控制台看自动化执行的日志：
    打开 `content.js`，将顶部的 `const ENABLE_LOG = false;` 修改为 `true`。
*   如果您希望保持控制台清爽：
    保持 `const ENABLE_LOG = false;` 即可。

## ⚠️ 避坑指南

1.  **上下文失效问题 (Context Invalidated)**：当您更新或重新加载插件时，旧页面中的定时器可能会失效并抛出 `Extension Context Invalidated` 异常。本插件在所有 API 调用前均加入了 `isContextValid()` 机制以实现无感销毁，杜绝崩溃。
2.  **点击穿透与事件触发**：游戏页面使用了虚拟 DOM，为防范假死，插件点击触发机制采用了完整的 `mouseover ➔ mousedown ➔ mouseup ➔ click` 事件序列来欺骗虚拟 DOM 树，并添加了点击冷却，防止快速连点报错。
